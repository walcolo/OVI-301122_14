INTRODUCCIÓN AL DISEÑO DE SITIOS WEB

Objetivo general:

Presentar el OVI con sus estructuras, generalidades y algunas temáticas generales del curso de Diseño de Sitios Web.

Objetivos del curso

Temáticas

Estimado (a) estudiante, sea bienvenido al Objeto Virtual de Información O.V.I. este objeto virtual esta dirigido a todos los estudiantes de la carrera Ingeniería de sistemas y estén interesados en profundizar en la administración y diseño de sitios Web. Presenta contenido de lectura y multimedia relacionada con la configuración y uso de GIT y GIHUB para la gestión de proyectos web y conceptos de HTML5 y CSS3 y su conceptualización y la importancia que tiene en la elaboración de sitios Web. Unidades  Instalación GIT y GITHUB  HTML5  CSS3. Dentro de las tres unidades abordadas encontraran lecturas que les dará las pautas necesarias para la hora de desarrollar proyectos de sitios Web.
